ソーシャルサービスをインストールするにはバージョン 29 以上の Firefox が必要です。今すぐお使いの Firefox を更新しましょう。ダウンロード.

このサービスを使用するには Firefox の開発版が必要です。

ホーム

Firefox に Cliqz が追加されました！

  1. Cliqz を通じて気に入ったコンテンツを共有

    2. ボタンをクリックするだけで、Cliqz を通じ、あなたのソーシャルネットワーク上であらゆるサイト上の興味深いコンテンツを共有できます。

  2. ツールバーへサイドバーボタンを追加

    3. Firefox 30 以降、「サイドバー」ツールバーボタンを使ってサイドバーを管理できるようになりました。「サイドバー」ボタンは カスタマイズパレット に含まれていて、このボタンをツールバーへドラッグして追加すれば、サイドバーが複数あっても素早く切り替えられます。

    さらに、サイドバー自体から、サイドバー内のサービスを変更したり、サイドバーを隠したりすることも簡単にできます。Firefox の 表示 > ツールバー メニューを使っても同じことが可能です。

  3. アドオンマネージャで設定を変更

    4. Firefox のアドオンマネージャを開けば、お使いのサービスを管理 したり、一時的に無効化したり、あるいは削除したりできます。

  4. 次の手順

    • このサービスが Firefox へ追加する機能について説明を見る
    • Cliqz が提供する機能についての 説明を見る
    • Firefox へ追加できる その他のサービスを見る
