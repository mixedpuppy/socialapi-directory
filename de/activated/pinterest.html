Firefox SocialAPI Demo Instructions

You are on a test version of this website. Some entries on this site are mockups and do not work as expected. If you did not intend to use the test site go to the production site.

To install from this directory you must modify the social.directories preference in Firefox to include the domain , you can do that in about:config.

Sie müssen Firefox verwenden, um soziale Dienste zu installieren.Herunterladen.

Installation von sozialen Diensten erfordert mindestens die Firefox-Version 29. Aktualisieren Sie jetzt Ihren Firefox.Herunterladen.

Dieser Dienst erfordert eine noch nicht veröffentlichte Firefox-Version.

Startseite

Herzlichen Glückwunsch, Pinterest ist jetzt in Firefox aktiviert!

  1. Teilen Sie Inhalte, die Sie mögen, über Pinterest

    2. Mit einem Klick können Sie über Pinterest interessante Inhalte auf einer beliebigen Website mit Ihrem sozialen Netzwerk teilen.

  2. Einstellungen über den Add-ons-Manager ändern

    3. Sie können einfach Ihre Dienste verwalten, sie temporär deaktivieren oder entfernen, in dem Sie den Add-ons-Manager in Firefox aufrufen.

  3. Nächste Schritte

    • Erfahren Sie mehr darüber, was dieser Dienst zu Firefox hinzufügt.
    • Erfahren Sie mehr darüber, was Pinterest anbietet.
    • Entdecken und finden Sie weitere Dienste, die Sie zu Firefox hinzufügen können.
