Mozilla

Sie müssen Firefox verwenden, um soziale Dienste zu installieren.Herunterladen.

Installation von sozialen Diensten erfordert mindestens die Firefox-Version 29. Aktualisieren Sie jetzt Ihren Firefox.Herunterladen.

Dieser Dienst erfordert eine noch nicht veröffentlichte Firefox-Version.

Startseite

mixi

Herzlichen Glückwunsch, mixi ist jetzt in Firefox aktiviert!

  1. Teilen Sie Inhalte, die Sie mögen, über mixi

    2. Mit einem Klick können Sie über mixi interessante Inhalte auf einer beliebigen Website mit Ihrem sozialen Netzwerk teilen.

  2. Fügen Sie Ihrer Symbolleiste die Sidebar-Schaltflächen hinzu

    3. Ab Firefox 30 können Sie mit der Sidebars-Schaltfläche in der Symbolleiste Ihre Sidebars verwalten. Die Sidebars-Schaltfläche befindet sich im Anpassen-Bereich, ziehen Sie einfach die Schaltfläche in Ihre Symbolleiste, um schnell zwischen Sidebars zu wechseln.

    Außerdem können Sie von der Sidebar selbst einfach den Dienst in Ihrer Sidebar wechseln oder die Sidebar ausblenden. Sie können auch das Menü Ansicht → Symbolleisten in Firefox verwenden.

  3. Einstellungen über den Add-ons-Manager ändern

    4. Sie können einfach Ihre Dienste verwalten, sie temporär deaktivieren oder entfernen, in dem Sie den Add-ons-Manager in Firefox aufrufen.

  4. Nächste Schritte

    • Erfahren Sie mehr darüber, was dieser Dienst zu Firefox hinzufügt.
    • Erfahren Sie mehr darüber, was mixi anbietet.
    • Entdecken und finden Sie weitere Dienste, die Sie zu Firefox hinzufügen können.
