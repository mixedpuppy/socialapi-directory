Firefox SocialAPI Demo Instructions

You are on a test version of this website. Some entries on this site are mockups and do not work as expected. If you did not intend to use the test site go to the production site.

To install from this directory you must modify the social.directories preference in Firefox to include the domain , you can do that in about:config.

Devi utilizzare Firefox per installare i servizi Social.Scarica.

L’installazione dei servizi Social richiede Firefox 29 o una versione successiva. Aggiorna subito la tua versione di Firefox.Scarica.

Questo servizio richiede una versione non ancora distribuita di Firefox.

  1. Aggiungi il pulsante Barre laterali alla barra degli strumenti

    2. A partire da Firefox 30 è possibile gestire le barre laterali utilizzando il pulsante Barre laterali direttamente dalla barra degli strumenti. Il pulsante è disponibile nella finestra di personalizzazione: trascina il pulsante nella barra degli strumenti per avere accesso in modo rapido a tutte le barre laterali disponibili.

    Inoltre, puoi cambiare il servizio disponibile o nascondere la barra laterale direttamente dalla barra stessa. Puoi anche utilizzare il menu di Firefox Visualizza -> Barre degli strumenti.

  2. Modifica le impostazioni dal Gestore componenti aggiuntivi

    3. Puoi gestire i tuoi servizi con estrema facilità, disattivandoli temporaneamente o rimuovendoli, direttamente nel Gestore componenti aggiuntivi di Firefox.
